# ML-Primes
Predicting prime numbers with machine learning algorithms

## Projects
### Predicting Primes
Given primes p_n, p_n+1, and p_n+2, predict the value of prime p_n+3.

### Factoring Primes
Given a product of two primes, predict the values of the factors.
